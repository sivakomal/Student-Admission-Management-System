# Student Admission Management System

## Overview

This repository contains a simple Student Admission Management System that allows students to input their academic details, personal information, and course preferences. The system also enables authorities to check and manage student details, respond to queries, and monitor the student status.

## 1. Student Menu

- Start the application by accessing the student menu.
- Navigate through the options to input academic details, personal information, and course preferences.

## 2. Academic Details

### 2.1 10th Class Details

- Input information related to 10th class.

### 2.2 Intermediate Details

- Input information related to intermediate studies.

## 3. Personal Information

- Input personal details such as name, address, and contact information.

### 3.1 Display Personal Details

- View and verify personal details entered.

## 4. Course Information

- Select a particular course and input relevant information.

### 4.1 Enter and Display Course Information

- Enter details related to the selected course and view the entered information.

## 5. Respond to Queries

- Authorities can respond to queries raised by students.

### 5.1 Enter and Display Responses

- Enter responses to student queries and view the responses.

## 6. University End

- Access features related to student management from the university end.

### 6.1 Check Student Status

- Check the status of a particular student.

### 6.2 Access Student Status Details

- View detailed information about student status.

### 6.3 Access Fee Details

- Check the fee details of students.

### 6.4 Enter and Display Fee

- Enter and view fee-related information.

## 7. Result

- After completing the necessary steps, the student will receive the result.

**Note:** This system is a simplified representation and may require further development and customization based on specific requirements.
